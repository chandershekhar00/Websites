# 📘 PassportSeva – UX Redesign Project

A **user-centered redesign** of India’s passport application portal focused on improving usability, accessibility, and overall user experience.

---

## 🚀 Project Overview

This project reimagines the passport application process by simplifying complex workflows, reducing user confusion, and providing a guided, step-by-step experience.

---

## 🎯 Objectives

* Simplify form filling
* Provide clear document guidance
* Improve appointment booking flow
* Reduce user anxiety
* Make the system accessible for all users

---

## ✨ Key Features

* ✅ **4-Step Form Wizard** (instead of long form)
* 💾 **Autosave System** (every 45 seconds)
* 📱 **OTP-Based Authentication** (no passwords)
* 📎 **Visual Document Upload Cards**
* 📅 **Integrated Calendar + Time Slot Booking**
* 📊 **User Dashboard with Status Tracking**
* 🎉 **Confirmation Screen with Timeline & Download Options**

---

## 🧠 UX Improvements

| Feature  | Before                 | After              |
| -------- | ---------------------- | ------------------ |
| Form     | 50+ fields in one page | 4-step guided form |
| Save     | No autosave            | Autosave enabled   |
| Upload   | Basic file input       | Visual card system |
| Booking  | Multi-page flow        | Single screen      |
| Feedback | Minimal                | Real-time feedback |
